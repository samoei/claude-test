# Tic-Tac-Toe

A simple two-player Tic-Tac-Toe game that runs in the browser — no installs or dependencies required.

## How to Play

1. Download or clone this repository
2. Open `tictactoe.html` in any web browser
3. Two players take turns clicking cells — **X** goes first
4. First to get three in a row (horizontally, vertically, or diagonally) wins
5. Click **New Game** to reset the board and play again

## Features

- Two-player local multiplayer (same screen)
- Scoreboard tracking wins for X, O, and draws across rounds
- Winning cells highlighted with a pulse animation
- Fully self-contained — a single HTML file with no external dependencies

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- No frameworks, no build tools
